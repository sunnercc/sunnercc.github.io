关键字的区别

readonly

  • 修饰任何类型，仅仅生成get方法

readwrite

  • 修饰任何类型，生成get方法，又生成set方法

strong

  • 修饰对象类型. 强引用被修饰对象

copy

  • 修饰不可变对象类型。强引用被修饰对象

weak

  • 修饰对象类型。弱引用被修饰对象，当被修饰对象释放的时候，weak修饰的对象指针会指向nil

unsafe_unretained (不safe不retain)

  • 修饰对象类型。弱引用被修饰对象。与weak相似。区别在于:当被修饰对象释放的时候，weak修饰的对象指针不会指向nil,会造成野指针。
